# OCR Text Extraction from Image

This project extracts text from images using Python and Tesseract OCR.

## Technologies Used
- Python
- pytesseract
- Pillow (PIL)
- Tesseract OCR
- VS Code

## Project Structure
- server.py – OCR logic
- test.png – Sample image
- app.py – Entry file
- *.ipynb – Practice notebooks

## How to Run
1. Install Tesseract OCR
2. Install dependencies:
   pip install pytesseract pillow
3. Run:
   python server.py

## Output
Extracted text from the image is returned as a string.
